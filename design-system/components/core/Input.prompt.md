Understated underline field for booking and enquiry forms; the line warms to sage on focus.

```jsx
<Input label="Nombre" placeholder="Tu nombre" />
<Input label="Fechas" hint="Llegada — salida" />
<Input label="Email" error="Correo no válido" />
```

Label renders as a quiet overline. Pass `error` to show a terracotta line + message.
