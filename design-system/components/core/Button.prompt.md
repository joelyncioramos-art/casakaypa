Quiet, editorial button with uppercase wide-tracked label; use for primary and secondary actions across Casa Kaypa surfaces.

```jsx
<Button variant="primary">Reservar</Button>
<Button variant="outline" size="lg">Ver la casa</Button>
```

Variants: `primary` (sage fill), `secondary` (terracotta fill), `outline` (ink hairline — the logo language, fills charcoal on hover), `ghost` (text with cream hover), `link` (inline underlined terracotta). Sizes `sm | md | lg`; `fullWidth` for stacked mobile CTAs. Buttons use the pill radius — the only place large pill radii appear.
