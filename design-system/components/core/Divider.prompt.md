The brand's signature hairline rule, drawn from the logo's line-art language.

```jsx
<Divider variant="short" />   {/* 48px terracotta accent under an eyebrow */}
<Divider />                   {/* full-width 1px ink rule */}
<Divider variant="soft" />    {/* subtle neutral hairline */}
```

Use `short` as a decorative accent, `default` to separate major blocks, `soft` for list rows.
