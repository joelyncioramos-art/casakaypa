Content container for rooms, experiences and stories; compose Eyebrow, headings, Tag and Button inside.

```jsx
<Card media="/assets/room.jpg" hoverLift>
  <Eyebrow>Suite</Eyebrow>
  <h3>La Casa del Árbol</h3>
  <p>Wake to birdsong and valley light.</p>
</Card>
```

Variants: `raised` (white + soft shadow), `outline` (ink hairline frame — the logo language), `sunken` (cream well). Set `hoverLift` for grids of clickable cards.
